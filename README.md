# Nexa AI

**Next Step fo the Future**

A production-ready full-stack AI Career Guidance + Learning + Certification Platform for students, professionals, and job seekers. Built with:

- React.js (Vite, Tailwind CSS, ShadCN UI, Framer Motion)
- Node.js, Express.js
- MongoDB Atlas
- JWT, Google OAuth, Email OTP
- OpenAI/Claude API
- Free course and certificate access

## Branding
- Uses Nexa AI logo and certificate style everywhere
- Blue gradient, gold, navy, white theme
- Poppins/Inter for UI, Times New Roman for certificates

## Structure
- `/frontend` — client app
- `/backend` — server API
